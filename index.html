BruteRaven

Ferramenta educacional para testes de força bruta em redes Wi-Fi. Apenas para fins educacionais e testes autorizados.

AVISO LEGAL: Este software é apenas para fins educacionais. Testar redes sem autorização é crime.

Recursos Técnicos

Escaneamento Avançado

Detecta redes Wi-Fi disponíveis com informações detalhadas sobre força do sinal e tipo de segurança.

  • Lista completa de redes visíveis
  • Indicador de força do sinal
  • Detecção de tipo de criptografia
  • Atualização em tempo real

Wordlists Flexíveis

Suporte a múltiplas fontes de wordlists para testes de força bruta.

  • Top100k (download automático)
  • RockYou (opcional)
  • Arquivos locais .txt
  • URLs remotas
  • Cache inteligente

Otimizações

Configurações avançadas para melhorar a eficiência dos testes.

  • Modo Verbose para detalhes
  • Modo Rápido (menos delay)
  • Forçar download de wordlists
  • Seleção específica por SSID
  • Interface limpa e intuitiva

Como Usar

Interface Gráfica

Execute python ui_launcher.py para iniciar a interface gráfica que irá escanear redes disponíveis e permitir configurar o ataque.

Recomendado

Linha de Comando

Para usuários avançados, execute diretamente python main.py com parâmetros personalizados.

# Exemplo básico:
python main.py -w top100k

Comandos Disponíveis

Parâmetro Descrição Exemplo
-w, --wordlist Seleciona wordlist (top100k/rockyou) -w top100k
-f, --file Usa arquivo local como wordlist -f lista.txt
-u, --url Baixa wordlist de URL remota -u http://exemplo.com/lista.txt
--ssid Especifica rede-alvo por SSID --ssid "MinhaRede"
-v, --verbose Ativa modo detalhado -v
--quick Ativa modo rápido (menos delay) --quick
--force-download Força novo download da wordlist --force-download

Aviso Legal

Uso Responsável

  • Não utilize esta ferramenta em redes que você não possui
  • Não utilize para atividades ilegais
  • Respeite a privacidade alheia
  • Consulte as leis locais sobre testes de segurança
